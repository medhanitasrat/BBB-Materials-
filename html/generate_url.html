generate_url#

Given a company information this tool will generate a potential URL :

Usage#

(.venv) $ pip install urllib
(.venv) $ pip install urlopen
(.venv) $ pip install search
(.venv) $ pip install requests
(.venv) $ pip install tldextract
(.venv) $ pip install ssl
generate_url.appends(company_name, tld, url=[])#

A funtion that appends company name to a general URL format

Parameters:

  • company_name – The name of the company

  • tld – top level domains

  • url – the list to store urls

Returns:

appended Urls

generate_url.clean_data(companyName)#

A function that cleans companyName(changes to lowercase, and all remove all white spaces as well as remove any legal designation eg: ‘,LLC’

Parameters:

companyName – The name of the company

Returns:

The name of the company after being cleaned

generate_url.validURl(url)#

A function that checks if found url is valid

Parameters:

url – The url

Returns:

True if the url is valid, false otherwise

generate_url.getURL(company_name, Url, rating_sites)#

Return company’s URL given company name and state

Parameters:

  • companyName – the name of the company

  • Url – a list where the URLs are stored

  • rating_sites – list of any known rating sites

Returns:

company’s URL if found, else return ‘’

generate_url.filter(url, rating_sites)#

A function that checks if found url are rating sites

Parameters:

  • url – the url

  • rating_sites – list of any known rating sites

Returns:

True if url is a not a rating site, false otherwise

generate_url.status_code(url)#

Gets a single url and returns the status code

Parameters:

url – a single url

Returns:

status code of the url if it receives a response within the given time, if not returns -1

generate_url.scrap(url, companyName, words)#

Scrapes website and checks if any of the company information exists within the website Adds one to count whenever an information is found within a website except if that information is company name, if that’s the case then adds 3 to count

Parameters:

  • url – The url

  • companyName – The company name

  • words – list if company information ( email, phone number, state, city, address, company name)

Returns:

the amount of how many urls have been found

generate_url.correctUrl(url, company_name, info)#

A function that picks the url with the most count out of the list of potential urls passed

Parameters:

  • url – the list of potential urls

  • company_name – the name of the company

  • info – company information

Returns:

the correct url or empty space of url not found

generate_url.company_URL(information)#

A function that gets the correct url utilizing all the funtions above

Parameters:

information – company information

Returns:

the correct url
